  1. Element选择：document.querySelector(‘html’)
  2. Element创建：document.createElement(‘button’)
  3. Element 添加：document.body.appendChild(Button)
  4. Element移除：Button.parentNode.removeChild(Input)
  5. Element赋值：document.querySelector(‘button’).textContent = ‘Hello world’;
  6. Element style: Button.style.backgroundColor=‘white’;
  7. Variable类型: String, Number, Boolean, Array, Object, Undefined, Null
  8. Event: document.querySelector(‘html’).onclick = function(){};
  9. Event监听：Button.addEventListener(‘click’, function(){})
  10. Prompt : var myName = prompt(“Please enter your name.”); console.log(“your enter name:”+myName)
  11. localStorage:本地储存 localStorage.setItem(); localStorage.getItem();
  12. sessionStorage:对话储存
