Angular Language Service

What is Angular Language Service

Angular Language Service是一种获取Angular模板中的完成，错误，提示和导航的方法，无论它们是HTML文件中的外部还是嵌入到字符串中的注释/装饰器中。 Angular Language Service会自动检测您打开Angular文件，读取您的tsconfig.json文件，找到您的应用程序中的所有模板，然后为您打开的任何模板提供语言服务。
详情以官方文档为主。

Step 1: Visual Studio Code Install Angular Language Service
        ext install ng-template or

        Click extensions button, find Angular Language Service, download...
Step 2: install @angular/language-service
        devDependencies {
                '@angular/language-service': '^5.0.0'
        }
Step 3: Add Typescript
        yarn add typescript@2.4.2
Step 4: Editer tsconfig.json (也可以不用添加) 
        'plugins': [
            {'name': '@angular/language-service'}
        ]
